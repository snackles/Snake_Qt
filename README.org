* Snake Qt
Classic Snake game developed using *C++20* and *Qt6*. The project utilizes *OOP* style and cross-platform compatibility through **CMake**.

** Installation & build
*** Need to install:
- CMake 4.2+ 
- Qt 6.x 
- C++20 compatible compiler (GCC/Clang/MSVC) 

*** Build steps
#+BEGIN_SRC bash
# Clone the repozitory 
git clone [https://github.com/snackles/Snake_Qt](https://github.com/snackles/Snake_Qt)
cd Snake_Qt

# Build
cd build
cmake ..
cmake --build .
#+END_SRC

*** Run
Linux / MacOS:
#+BEGIN_SRC bash
./Snake
#+END_SRC
Windows:
#+BEGIN_SRC bash
Snake.exe
#+END_SRC

** Controls:
| Key             | Action                               |
|-----------------+--------------------------------------|
| *W / A / S / D* | Move (Up, Left, Down, Right)         |
| *P*             | Pause / Resume                       |
| *R*             | Restart (anytime or after Game Over) |
| *ESC*           | Quit game                            |

** Project structure:
- =src/= :: implementation files (.cpp)
- =include/= :: header files (.hpp)
- =assets/= :: game assets
- =CMakeLists.txt= :: build configuration script
- =resources.qrc= :: .qrc file
